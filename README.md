混淆文档说明

代码由纯python编写完成，在运行前需要进行的操作包括:

1.安装python3

1）若已安装过brew，可在命令行直接输入brew install python3;

2）可直接到官网进行安装

安装完成后在终端输入：python3查询是否安装完成

2.运行环境

1）若已有pycharm，可在pycharm内直接运行

2）若无pycharm，在终端直接运行py文件

脚本实现的功能

1.类名混淆;

2.方法名与属性名混淆;

3.方法内参数混淆和局部变量名混淆;

4.无效代码的插入;

5.垃圾属性传值赋值;

6.串联垃圾方法;

7.调整代码运行逻辑;
...
